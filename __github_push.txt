## Examples, Hints, Reminders

* Bash commands ('first time'): 
```
ssh-keygen -t rsa -b 4096 -C "<your-github-username>@users.noreply.github.com" -f ~/.ssh/id_rsa_github-dev
cat ~/.ssh/id_rsa_github-dev.pub
# copy + paste this for new Github key (profile icon => Settings => SSH + GPG keys)
```

* Bash, adding personal SSH identity ('always'): 
```
cd /var/www/html/_fh/lib4ri_solr_config
eval $(ssh-agent -s)
ssh-add ~/.ssh/id_rsa_github-dev
ssh -vT git@github.com
```

* Push onto Github ('first time'): 
```
cd /var/www/html/_fh/lib4ri_solr_config
git init
git add .
git commit -m "Support for citation related datastream"
git branch -m master 4.x
git remote add origin git@github.com:Lib4RI/lib4ri_solr_config.git
git remote set-url origin git@github.com:Lib4RI/lib4ri_solr_config.git
git push -u origin 4.x
```

* Push onto Github ('consecutive'): 
```
cd /var/www/html/_fh/lib4ri_solr_config
git init
git add .
git commit -m "Support for citation related datastream"
git push -u origin 4.x
```
